# 👑 RoyalPartner Website v2.0

Sito web professionale per la gestione di Partnership Discord e Servizi di configurazione server.

## 🚀 Caratteristiche
- **Next.js 14 App Router**: Per prestazioni fulminee.
- **Discord Webhook Integration**: Le richieste degli utenti arrivano direttamente sul tuo server.
- **Admin Dashboard**: Interfaccia di gestione per lo staff.
- **Tailwind CSS**: Design ultra-moderno e reattivo.
- **Framer Motion**: Animazioni premium.

## 🛠️ Installazione
1. Clona la repository.
2. Esegui `npm install`.
3. Crea un file `.env.local` e inserisci `NEXT_PUBLIC_DISCORD_WEBHOOK_URL`.
4. Avvia con `npm run dev`.

## 📦 Deployment
Il sito è ottimizzato per **Vercel**. Basta collegare la repo GitHub a Vercel e aggiungere la variabile d'ambiente del Webhook.
